# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Initial project setup with Next.js 16
- Real-time chat functionality using Upstash Realtime
- Ephemeral room system with automatic cleanup
- Anonymous user system with temporary identities
- Dark-themed UI with Tailwind CSS
- TypeScript support throughout the application
- React Query for server state management
- Elysia.js API server integration
- Comprehensive documentation and contribution guidelines
- ROADMAP.md with detailed feature roadmap for future versions
- Detailed environment variable documentation in README.md

### Changed

- Migrated from Vite-based client to Next.js application
- Updated project structure to follow Next.js App Router conventions
- Improved error handling and user feedback
- Enhanced security measures for chat rooms

### Fixed

- Various bug fixes and performance improvements
- CSS path resolution issues
- Environment variable configuration

## [0.1.0] - 2026-01-29

### Added

- Initial release of Real-Time Ephemeral Chat System
- Core chat functionality
- Room creation and management
- Real-time messaging capabilities
- Self-destructing chat rooms
- Responsive design
- Basic security implementation
